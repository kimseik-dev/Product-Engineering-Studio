# 🚀 Product Engineering Studio Dashboard

대표님! 이 프로젝트는 **Vite + React** 기반의 프리미엄 대시보드 공간이에요! 💖
여기에 저 영자가 특별히 장착해 드린 **"Skill System"** 사용법을 정리했으니 확인해 보세요! 얍! 🌈

---

## 🎨 영자의 스킬 가이드 (Skill System)

이제 저 영자가 대표님의 아이디어를 더 멋지게 실현할 수 있도록 **1,000개 이상의 스킬**을 자유자재로 찾아보고 장착할 수 있게 되었어요!

### 🔍 스킬 찾기 (Find Skills)
나중에 저한테 이렇게 물어보시면 돼요! (또는 터미널에 직접 입력하셔도 된답니다 😉)

```bash
# 리액트 관련 스킬 찾기 예시
npx skills find react
```

- **저한테 물어보실 때**: "영자야, 애니메이션 스킬 좀 찾아줘~"
- **명령어로 직접 쓰실 때**: `npx skills find [키워드]`

### 🛠 스킬 추가하기 (Add Skills)
마음에 드는 스킬을 발견했다면? 바로 장착!

```bash
npx skills add [스킬경로] -g -y
```

### 📦 스킬 보물창고 (Awesome Skills)
프로젝트 내의 `./antigravity-awesome-skills` 폴더에 1,000개가 넘는 다양한 스킬들이 모여있어요! 저 영자가 여기서 대표님께 꼭 필요한 보물 같은 스킬들을 쏙쏙 골라드릴게요! ✨
